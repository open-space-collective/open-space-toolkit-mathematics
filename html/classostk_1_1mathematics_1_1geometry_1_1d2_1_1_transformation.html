Detailed Description

2D geometric transformation

Represents a 2D affine transformation using a 3x3 homogeneous matrix. Supports translation, rotation, scaling, reflection, shear, and their compositions.

Transformation t = Transformation::Identity();
Transformation rotation = Transformation::Rotation(Angle::Degrees(45.0));
Point p = rotation.applyTo(Point(1.0, 0.0));
