Creates a new array that contains the gaps between an array of intervals. If an interval is actually defined, only intervals intersecting with the interval are considered and potential leading and trailing gaps w.r.t. the interval boundaries are included. The intervals are previously merged and sorted.

Array<Interval<Real>> intervals = {
Interval<Real>::Closed(-1.0, 1.0),
Interval<Real>::Closed(2.0, 4.0),
};
const Interval<Real> bounds = Interval<Real>::Closed(1.5, 4.5)
Array<Interval<Real>> gaps = Interval<Real>::GetGaps(intervals, bounds);
// {[1.5, 2.0], [4.0, 4.5]}
Returns
Gaps array
