    Class ostk::mathematics::geometry::d3::Transformation¶

    class Transformation¶

    3D geometric transformation

    Represents a 3D affine transformation using a 4x4 homogeneous matrix. Supports translation, rotation, scaling, reflection, shear, and their compositions.

    Transformation t = Transformation::Identity();
Transformation rotation = Transformation::Rotation(RotationVector({0.0, 0.0, 1.0},
Angle::Degrees(45.0))); Point p = rotation.applyTo(Point(1.0, 0.0, 0.0));
