    class CubicSpline : public ostk::mathematics::curvefitting::Interpolator¶

    CubicSpline.

    A cubic Spline interpolator is a spline where each piece is a third-degree polynomial specified in Spline form, that is by its values and first derivatives at the end points of the corresponding domain interval.

    https://en.wikipedia.org/wiki/Cubic_Spline_spline#:~:text=In%20numerical%20analysis%2C%20a%20cubic,of%20the%20corresponding%20domain%20interval.
